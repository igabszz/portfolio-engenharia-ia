# 🚀 Engenharia de IA & Dados - Portfólio Profissional

Bem-vindo ao meu repositório de soluções inteligentes. Aqui aplico minha base sólida em **Administração de Bancos de Dados (DBA)** para construir agentes de IA e pipelines de processamento de dados de alta performance.

## 📂 Projetos em Destaque

### 1. [Agente SQL (Text-to-SQL)](./agente-ia-sql/)
* **O que faz:** Interface em linguagem natural para bancos de dados relacionais.
* **Diferencial:** Utiliza LangChain e Gemini 3 Flash para permitir que usuários não técnicos consultem dados complexos sem saber SQL.
* **Foco:** Governança de dados e agilidade analítica.

### 2. [Vision AI & OCR Inteligente](./visao-computacional/)
* **O que faz:** Pipeline de **Intelligent Document Processing (IDP)**.
* **Diferencial:** Extração de dados estruturados (JSON) a partir de imagens de documentos (fichas, contratos, notas).
* **Foco:** Visão Computacional e automação de processos manuais.

## 🛠️ Stack Técnica
* **Linguagens:** Python (Automação e IA), SQL (Oracle, SQL Server, PostgreSQL).
* **IA & LLMs:** LangChain, Google Gemini 3 Flash, OpenAI, Prompt Engineering.

Demonstrar a aplicação prática de IA Generativa para resolver problemas reais de negócios, com foco em escalabilidade, segurança e resolutividade.
